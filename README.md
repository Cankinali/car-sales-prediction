Araba Satış Tahmini Projesi / Car Sales Prediction
Proje Hakkında / About
Araç özelliklerine dayalı satış tahmini yapan makine öğrenmesi projesi. Fiyat, motor hacmi, beygir gücü ve yakıt verimliliği gibi parametreler kullanılarak gelecekteki satış miktarları tahmin edilmiştir.
A machine learning project that predicts car sales based on vehicle features such as price, engine size, horsepower, and fuel efficiency.
Kullanılan Teknolojiler / Technologies

Python, Pandas, Scikit-learn
Linear Regression, Random Forest, Gradient Boosting
Matplotlib, Seaborn

Özellikler / Features

Veri temizleme ve ön işleme
3 farklı ML modeli karşılaştırması
Korelasyon ve görselleştirme analizi
Yakıt verimliliği segmentasyonu
Performans metrikleri (MAE, MSE, R²)

Sonuçlar / Results
Ensemble modeller (Random Forest, Gradient Boosting) lineer regresyona göre daha iyi performans gösterdi. Fiyat-yeniden satış değeri korelasyonu: 0.95
